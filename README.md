## TacheMun
Gestionnaire de Taches en Android

- Affichage de liste des tâches
- Organisation des taches en listes
- Création / modification / suppression de tâches 
- Systeme de rappel/alerte
- Seadline pour les tâches (gestion de dates )
- Travailler à plusieurs, toutes sous tâches -> assigner des tâches
- Statut des tâches/état d'avancement
- Gestion des priorités de tâche
- Partage de tâches avec d'autres applications ( représentation textuelle par email)
- Partage par Bluetooth
- Mise en place d'un service centralisé (bdd externe)
- Gestion d'authentification ( Facebook/google ) 
- Base de données pour les stockages locaux (mode hors ligne)
- Exposition des données via un content provider (lecture/écriture)
- Ajouter un lien avec géo-localisation
